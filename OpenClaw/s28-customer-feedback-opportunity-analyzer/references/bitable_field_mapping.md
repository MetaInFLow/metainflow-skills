### 日期字段格式

- **输入格式**：`YYYY/MM/DD` 或 `YYYY-MM-DD`
- **API 格式**：毫秒级时间戳（飞书多维表 API 要求）
- **转换示例**：
  ```python
  from datetime import datetime
  dt = datetime.strptime("2026/03/15", "%Y/%m/%d")
  timestamp_ms = int(dt.timestamp() * 1000)
  ```

### 人员字段

- **输入格式**：用户姓名（需映射）或 open_id
- **API 格式**：飞书用户 ID（需配置用户映射）

### 关联字段

- **输入格式**：关联记录的 record_id 或通过查找获取
- **API 格式**：飞书多维表记录 ID

## 注意事项

1. **字段名称**：写入字段名必须与多维表中的中文字段名保持一致
2. **必填字段**：创建记录时必须填写所有必填字段
3. **日期转换**：日期字段优先使用 13 位毫秒时间戳
4. **关联字段**：`客户名称`、`项目编号`、`关联客户` 等关联字段必须传入记录 ID 数组
5. **人员字段**：人员字段需要真实用户 ID，不能直接写姓名
6. **自动编号**：反馈编号由多维表自动生成，创建记录时无需填写
7. **表纠偏**：你提供的共享链接落在 `1- 线索管理表`，真正的反馈写入表是 `客户反馈表`
